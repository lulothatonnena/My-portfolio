Project 1

To-Do List Application

This To-Do List application is a simple yet powerful tool designed to help users manage their tasks efficiently. Built using HTML, CSS, and JavaScript, the application allows users to add, edit, and delete tasks, as well as mark them as complete. It also includes filters to view all tasks, only active tasks, or only completed tasks.

Features:
  • Add Tasks: Users can easily add new tasks to their list.
  • Edit Tasks: Tasks can be edited to update their content.
  • Delete Tasks: Users can remove tasks that are no longer needed.
  • Mark as Complete: Tasks can be marked as complete, and they will be visually distinguished from active tasks.
  • Filter Tasks: Users can filter tasks to view all tasks, only active tasks, or only completed tasks.
Technologies Used:
  • HTML: For the structure of the application.
  • CSS: For styling and layout.
  • JavaScript: For interactivity and functionality.
